Udacity Reflection

What is grid based design?

Grid based design is basically dividing up your website page into a grid to
achieve the structure you want in your website.

Whats all the hype about responsive webpages?

Well if your web page isn't responsive then it may be difficult for your
website to be viewed on other devices. For example if your web page is designed
to fit on a regular computer screen and isn't responsive it may look jumbled on
an I phone.

What is semantic structure?

Having a semantic structure means you have semantic tags in your code. An
example of a semantic tag is h1 h2 h3 etc... These are titles.
